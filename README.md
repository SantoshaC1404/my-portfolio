A modern, responsive personal portfolio website built with **React.js** and **Tailwind CSS**, showcasing my skills, experience, projects, and contact information.

## ✨ Features

- ⚡ **Fast & Responsive** — Optimized for all screen sizes (mobile, tablet, desktop)
- 🎨 **Smooth Animations** — Parallax tilt effects, typewriter effect, and hover interactions
- 📂 **Projects Showcase** — Modal-based project viewer with GitHub and live demo links
- 🛠️ **Skills Section** — Visual skill icons grouped by category
- 💼 **Experience & Education** — Clean timeline layout
- 📬 **Contact Form** — Integrated with EmailJS for direct email sending
- 🌙 **Dark Theme** — Sleek dark UI with purple accent colors

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or above)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/SantoshaC1404/my-portfolio.git
   cd my-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
```
